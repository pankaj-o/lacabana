Image Placeholder Guide for La Cabana Website
==============================================

This folder contains placeholder images for the website. 
Replace these SVG files with actual high-quality images:

1. logo.svg - Restaurant logo (200 x 80)
2. hero-background.svg - Main hero section background (1920 x 1080)
3. restaurant-interior.svg - Interior photos (800 x 600)
4. tapas-dish.svg - Food photography (600 x 400)
5. wine-glass.svg - Wine and beverages (400 x 600)
6. event-space.svg - Event space photos (1200 x 800)

Recommended Image Formats:
- JPG for photographs
- PNG for logos and graphics with transparency
- WebP for modern browsers (better compression)

Recommended Sizes:
- Hero images: 1920 x 1080 or larger
- Section images: 1200 x 800
- Thumbnails: 400 x 300
- Logo: SVG or PNG with transparency

Image Optimization:
- Compress images before uploading
- Use appropriate resolution for web (72-150 DPI)
- Consider lazy loading for performance
